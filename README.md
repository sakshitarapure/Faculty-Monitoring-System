# Faculty Monitoring and Timetable Management System

## 📌 Project Overview
The Faculty Monitoring and Timetable Management System is a web-based application developed to simplify and automate faculty timetable management within an educational institution. The system allows teachers to manage timetable entries, enables HODs to review and approve/reject timetables, and helps principals monitor faculty schedules efficiently.

The project also integrates a chatbot to assist users with navigation and answering common queries related to the system.

---

## 🚀 Features

### 👨‍🏫 Teacher Module
- Add timetable entries
- Edit timetable entries
- Delete timetable entries
- View timetable in grid format
- Submit timetable for approval

### 🏢 HOD Module
- View faculty timetables
- Approve or reject timetable entries
- Add remarks/comments
- Monitor timetable status

### 👨‍💼 Principal Module
- Search faculty by name
- View faculty timetable
- Monitor current class details

### 🤖 Chatbot Module
- Botpress chatbot integration
- User assistance and navigation support
